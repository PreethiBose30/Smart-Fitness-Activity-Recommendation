# Smart Fitness Activity Recommendation System

## Problem Statement
Many individuals struggle to choose safe and appropriate fitness activities based on their physical condition, health status, and lifestyle. Incorrect or uninformed exercise choices may lead to injuries or ineffective fitness outcomes.

This project aims to develop a web-based intelligent fitness recommendation system that collects basic user health and physical data and suggests suitable fitness activities, diet guidance, and safety-aware recommendations.
