# Smart Fitness Activity Recommendation System

## Problem Statement
Many people are unsure about which fitness activities are safe and suitable based on their
medical and physical conditions. This project aims to develop a web-based system that
collects basic user medical data and provides appropriate fitness activity recommendations.

## Project Type
Multi Disciplinary Project (MDP)
